Este projeto em Python implementa um sistema simples de self-service para restaurantes, permitindo que os clientes escolham seus ingredientes e vejam as informações nutricionais do prato montado ao fim da seleção de ingredientes. O sistema é útil para restaurantes que desejam oferecer mais transparência e flexibilidade aos seus clientes. Saiba Mais!
